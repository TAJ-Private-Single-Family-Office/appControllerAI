# Microservices Architecture

## Overview
A microservices-based architecture for full-stack development using Azure and GitHub Enterprise Cloud.

### Components
- **Backend Services**: Individual services for accounts, payments, transactions.
- **Frontend**: React-based single-page application.
- **API Gateway**: Centralized entry point for microservices.
- **AI Services**: Using Azure AI Agent Service for analytics.

## Design Patterns
- **Domain-driven design (DDD)** for service boundaries.
- **Event-driven architecture** with Azure Event Grid.
- **API Gateway pattern** with Azure API Management.
